# Copywriting Visual Power (短视频文案改写技能)

这是一个基于 OpenClaw 规范的短视频文案改写与诊断技能。

## 简介
帮助用户把抽象、官方、没有感染力的文案，改写成有画面感、有力量感、让观众一句话就能代入的短视频文案。

## 核心功能与工作流
1. **诊断原文**：识别感受型、抽象型、缺细节、需对比的句子。
2. **定位关键句**：集中修改开头钩子句与核心论点句。
3. **改写强化**：叠加描述法、类比法、强弱对比与换动词。
4. **指标验收**：严格执行 7 条验收标准（如“村口老老太听得懂”、“脑海有画面”等）确保内容质量。

## 使用方法
将本目录放置在你的技能库中（如 `.gemini/config/skills/copywriting-visual-power`），或配合你的 OpenClaw 助手使用。
