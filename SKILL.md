# 短视频文案改写技能

帮助用户把抽象、官方、没有感染力的文案，改写成有画面感、有力量感、让观众一句话就能代入的短视频文案。

---

## 核心原则

> 好的文案，就是一个特写镜头。不是写感受，而是让观众"看见"。
> 

短视频用户在快速刷屏，一句话没画面、没代入感，就会划走。改写目标：**让村口老太太也能听懂，且脑海里能出现画面。**

---

## 改写工作流

### 第一步：诊断原文

逐句判断，把问题句子分类：

| 问题类型 | 判断标准 | 标记方式 |
| --- | --- | --- |
| 感受型 | 说的是情绪/主观感受，看不见具体场景 | 标记「感受」 |
| 抽象型 | 用了大词、官方腔、广告语 | 标记「抽象」 |
| 无细节型 | 意思能懂，但缺少人/物/动作/场景 | 标记「缺细节」 |
| 重点不突出 | 核心信息没有参照，显得普通 | 标记「需对比」 |

**诊断完成后，先告诉用户哪几句有问题、是什么问题，再开始改写。**

---

### 第二步：找关键句

> 不是每句话都需要全力改写。集中火力在关键句，其余句子够用就不动。
> 

**关键句通常是以下两类：**

- **开头钩子句**：第一句话，决定用户要不要继续看
- **核心论点句**：整条文案最想让观众记住的那一句

**每条文案找 1-2 句关键句，重点叠加改写方法。其余句子只做基础诊断，有明显问题再改，没问题就保留原来的节奏和语气。**

原因：全文每句都很强，等于全文都不强。文案需要有轻重节奏，强的地方才显得强。

---

### 第三步：改写

**关键句：叠加四个方法逐层强化**

四个方法作用在不同层面，可以同时叠加在同一句话上：

| 方法 | 作用层面 | 用法 |
| --- | --- | --- |
| 描述法 | 把感受变画面 | 把情绪/感受换成具体动作、场景、物品 |
| 类比法 | 把抽象变具体 | 用熟悉的事物替代难懂的概念或数字 |
| 强弱对比 | 让重点更突出 | 加参照物，从数量/时间/经历/身份四个方向对比 |
| 换动词 | 让语气更有劲 | 把软动词换成更有冲击力的词 |

对关键句，**依次过四个方法**，每个方法都问：这里能用吗？能用就加，不适用就跳过，但每个方法都必须判断，不能直接跳过不想。

---

**类比法参考示例库**

找类比对象的核心逻辑：用观众生活里熟悉的、能立刻想象出大小/重量/感觉的事物来替代。

| 想表达的意思 | 类比对象 | 改写后 |
| --- | --- | --- |
| 家很大 | 足球场 | 我家大到能在家里踢足球赛 |
| 脑袋很大 | 西瓜 | 脑袋跟西瓜一样大 |
| 某人很帅 | 彭于晏 | 长得像彭于晏那个方向，观众马上能感受到"帅"的程度 |
| 减肥成功 | 饮水机一桶水 | 三个月减了25斤，相当于减掉一整桶饮水机的水 |
| 工程车很大 | 车轮 and 人一样高 | 前面两个轮子跟我一样高，整辆车自然显得非常大 |
| 某人极其聪明 | 智商差距 | 狗智商约40，普通人约100，他168——他看我们，就像我们看狗 |
| 宝宝皮肤很嫩 | 剥壳鸡蛋 | 像剥了壳的鸡蛋，弹弹的，一摁好像能挤出水 |

**类比对象选择原则**：越日常越好，越具体越好。足球场、饮水机、鸡蛋——不是因为高级，而是因为人人见过、秒懂大小。

---

**换动词参考词库**

把平淡的动作词换成更有冲击力的词，让画面更有力量感。

| 普通动词 | 替换词 | 力量感来源 |
| --- | --- | --- |
| 打你 | 捶你 / 揍你 | 动作更重，听感更强 |
| 推一下 | 怼过去 | 更有冲击感 |
| 拉着他 | 薅着他 / 扯着他 | 连续用力，画面感更强 |
| 踢一下 | 踹一下 | 力量更明显 |
| 吃饭 | 造饭 / 炫饭 | 口语节奏更强 |
| 追着他跑 | 撵着他跑 | 追的状态更形象 |
| 跑得很快 | 撩得很快 | 速度感更强 |
| 使劲往里按 | 怼进去 | 一个字，动作感强 |
| 一直拽着用力 | 薅着不放 | 连续发力的画面感 |
| 往下扣 | 抠下来 | 方向感和力度都更清晰 |

**节奏加强词**：库库 / 哐哐 / 嗷嗷 / 咔咔——放在动词前后，加强强度与节奏感。
例：哐哐一顿打 / 咔咔往里怼 / 嗷嗷往外薅

**叠加示例：**

原句：我很努力，终于还清了债务。

- 描述法：还清那天，我把最后一张转账截图截图发给我妈
- 类比法：1000万，相当于还了我老家100套房
- 强弱对比：我爸那年工资才680，我背了他40年的工资
- 换动词：我把欠条一张一张撕掉，撕一张哭一声

四个维度同时作用，是四层不同的加强，不是互斥选项。

---

**非关键句：基础诊断，够用不动**

非关键句只做一件事：有明显问题就改，没问题就保留。

改写原则：**保留原作者的语气和节奏**，只修掉真正影响理解或代入的问题句。过度改写会让文案失去原来的风格，帮别人改稿时尤其注意。

---

### 第四步：验收

改写完成后，过以下五条标准。每条都要主动判断，不能跳过。

| # | 验收问题 | 没过怎么办 |
| --- | --- | --- |
| 1 | **村口老太太能听懂吗？** | 还有抽象词、大词、官方腔 → 继续改 |
| 2 | **脑海里有画面吗？** | 还是感受型表达 → 用描述法或类比法替换 |
| 3 | **大声读一遍，哪里卡了？** | 句子太长、太绕、停顿不自然 → 断句或重写 |
| 4 | **读完第一句，想继续看吗？** | 没有好奇感、共鸣感或爽感 → 钩子不够，重写开头 |
| 5 | **还有能删的句子吗？** | 删掉之后意思没少 → 那句话就是多余的，直接删 |
| 6 | **最强的一句，周围在托它吗？** | 周围句子和它抢强度 → 把非关键句调弱，让关键句突出来 |
| 7 | **读完最后一句，这件事说完了吗？** | 结尾没有落点 → 补一个结论、反转、或让人回味的画面 |

五条全部过了，这条文案才算合格。

## 常见错误提醒

- **不要全文每句都强改**：节奏需要有轻有重，否则全强等于全不强
- **不要改掉原作者的语气**：帮别人改稿时，风格是对方的，问题才是你要解决的
- **不要保留古诗词、文言文、广告腔**（\"岁月如歌\"\"砥砺前行\"这类全删）
- **不要堆砌感受词**（\"震惊\"\"感动\"\"绝望\"这类词能删就删，换成画面）
- **不要让方言成为理解门槛**（方言是加分项，不是必选项）
- **不要让重点淹没在平铺直叙里**（找到最核心的一句，用对比托出来）

## 快速判断口诀

```
是感受 → 描述法
太抽象 → 类比法
重点弱 → 强弱对比
语气软 → 换动词
啥都行 → 先问\"村口老太太能听懂吗\"
```
